# fkall  
Kill or force kill all OS X applications with processes containing input  

## How to install:  
Make a bin folder if needed:  
`mkdir ~/.bin`  
Move to bin folder:  
`cd ~/.bin`  
Download kall and fkall from github to the bin folder of your choice:  
`curl -O https://raw.github.com/pro-tester/fkall/master/kall`  
`curl -O https://raw.github.com/pro-tester/fkall/master/fkall`  
Add bin folder to PATH:
`echo 'export PATH=$PATH:~/bin\n' >> ~/.bash_profile`
